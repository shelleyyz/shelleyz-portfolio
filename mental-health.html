UX Design: Spotlight

Spotlight is a potential training solution designed to be a digital resource for adults diagnosed with attention deficit hyperactive disorder. This was a solo project and involved UX research including interviewing and synthesis of research and testing feedback. This was for a short course over 4 days. This is just the beginning.

Opportunity statement

"How might we provide a potential solution or approach for adults diagnosed with ADHD so that they feel supported in their social, work and personal lives?"

There are four key pain points:

  • Recognizing the disorder
  • Getting a diagnosis
  • Accessing treatment
  • Difficulties with ongoing life management

Next steps

I chose to tackle the last point for this project and have designed a prototype that is currently in testing via the Marvel app. You can view it here.

  • Conduct additional interviews and surveys for secondary stakeholders - ADHD coaches and clinics
  • Write user stories for basic functionality - registration, user profile, module features and interactivity
  • Complete wireframes and UI design for one module
  • Test it with users

Development: Rezbot

When you need a 'rez' in life. Rezbot is a blogging application that I built featuring a chatbot. Rez will direct you to relevant posts written by other users based on what you ask it.

    Functionality
  • Sign up as a new user or sign in if you're an existing user
  • View other profiles
  • View public posts
  • View your own private posts
  • Filter posts by categories
  • Add comments to posts
  • Chat to Rez
  • Rez will be able to direct you to the public posts page with filters applied based on what the user chooses to type such as posts relating to depression or anxiety.

Next steps

This doesn't look the best but I'm happy I was able to build what I had envisioned functionality-wise! You can view the source code here. I'd like to redo this project when I have time, focusing on:

  • The styling and design
  • Interactivity
  • Investigate dialogflow API so I can improve Rez's conversational design
